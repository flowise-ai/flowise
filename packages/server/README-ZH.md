 | 中文

<h3>以可视化方式构建 AI Agents</h3>



## ⚡ 快速入门

1. 安装 Flowise
    ```bash
    npm install -g flowise
    ```
2. 启动 Flowise

    ```bash
    npx flowise start
    ```

3. 打开

## 🌱 环境变量

Fl

您还可以在使用`npx`时指定环境变量。例如：

```
npx flowise start --PORT=3000 --DEBUG=true
```
